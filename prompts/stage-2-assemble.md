# Промпт для STAGE 2: Article Assembly

## Контекст
Ты собираешь 7-12 эпизодов в ОДНУ ЦЕЛУЮ статью.
НЕ копируешь эпизоды. Переписываешь их, добавляя переходы и глубину.

## Части статьи

### LEDE (начало): 600-900 знаков
- Крючок с эмоцией
- НЕ объяснение, а ДЕЙСТВИЕ/ДИАЛОГ
- Вопрос, который читатель будет держать в голове
- Пример: "Меня называли худшей невесткой. При всех. 12 лет."

### DEVELOPMENT (развитие): 1500-2000 знаков
- Наращивание напряжения
- Показать прогресс (недели, месяцы)
- "Первый месяц...", "К месяцу третьему..."

### CLIMAX (кульминация): 1200-1600 знаков
- Главная сцена конфронтации
- Антагонист ВИДИТ её успех
- ВИДИМАЯ реакция антагониста (рот открыт, дрожат руки)
- "Её лицо побелело"

### RESOLUTION (развязка): 1000-1300 знаков
- Её новая позиция (ТВЁРДО, без сомнений)
- Последствия для антагониста
- "Теперь она работает на меня"

### FINALE (финал): 1200-1800 знаков
- CAPS фраза с победой
- "Я НЕ ПРОСТО ВЫИГРАЛА. Я СТАЛА ИХ КОРОЛЕВОЙ."
- Вопрос к читателю (без ответа)
- Мощное завершение

## Правила
1. Не копировать текст эпизодов, переписать
2. Синхронизировать с PlotBible (имена, возраст, тон)
3. Переходы плавные (не скачки)
4. Каждая часть — НОВОЕ содержание, не компиляция

## OUTPUT
15,000-25,000 знаков готовой статьи
