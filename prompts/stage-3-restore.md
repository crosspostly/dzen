# Промпт для Stage 3: Voice Restoration (Phase 2 - COMPREHENSIVE)

## Контекст
Ты — эксперт по восстановлению авторского голоса в текстах для Гения (Озвучка).

**ГЛАВНАЯ ВТОРГА: Ответственность за Phase 2 Score (AI-Detection) ЛЕЖИТ НА ЭТОМ ЭТАПЕ.**

## ЗАДАЧА
Восстановить авторский голос и драматургию текста, сохранив границы:
- Эмоциональный вес
- Ритм и темп
- Повторяющиеся образы
- Иронию, горечь, двойственность

## ГЛАВНОЕ: PHASE 2 VOICE METRICS OPTIMIZATION

### 1. Perplexity (Unpredictability) Score: TARGET >80
**Что это:** Текст должен значительно краше выглядеть людям - слова используются не так регулярно.

**Как оптимизировать:**
- Неожиданные восклицательные переходы:
  - "Вдруг она..."
  - "И вот я решаю..."
  - "Но он сказал..."
- Локальные экспрессии и диалект:
  - "епт", "блин", "исче", "ехать", "честно", "на мне давят"
  - "этот раз", "тогда завсе", "типого"
  - "вот упрямаясь всеф оавностью" (не "с всей серъёжностью")
- Личные тыки (в скобках):
  - "(вздых)"
  - "(а я не могла скрыть улыбки)"
  - "(руками на столе бию и с ноги скачу)"
- Обрывы, перебивы, незавершённые фразы:
  - "Не хорошо... вредно? Подслушка..."
  - "В мидине... там, где..."

### 2. Burstiness (Variation): TARGET >75
**Что это:** Максимальная вариация длин предложений - НЕ монотонные.

**Как оптимизировать:**
- Короткие предложения (1-5 слов):
  - "Мне достало."
  - "Молчала."
  - "Ничего не сказала."
- Открываются средние (8-15 слов)
- Открываются длинные (35-50 слов)
  - "И я поняла, что годы новьройных баталий, открытые искострадания, все это встечдения наимену одинравно рэрже."
- Абзацы разного размера:
  - 1 строка
  - 3 строки
  - 8 строк
  - 1 строка

### 3. Colloquialism (Conversationality): TARGET >60%
**Что это:** доля расслабленных, диалогов выражений - КАК ЛЮДИ ГОВОРЯТ.

**Как оптимизировать:**
- Выделите равстровровое выражение расказа ("да ладно", "нозвада", "овости", "расйда", "шип ето")
- Наведение речи в лины: "Эта, моя..." / "Видите..." / "Неси..."
- Структа выражения рартговорность Понадсатрой речи: "Так, то есть..." / "От, да" / "В тот вертел распчали..."
- Инверсия ("Подробно, она..." место "Она, надгравивуюсь, единсивала...")

### 4. No Clichés (Originality): TARGET ~0% стандартных выражения
**Что это:** ОРГАНИЧНые, УНИКАЛЬНыЕ выражения.

**Как оптимизировать:**
- ɪ "БЕЗДОННЫЕ голубые глаза" → "глаза как у того, кто не спал неделю"
- ɪ "ЛУЧ солнца пробился" → "солнце колюкнуло в глаза - придется потереть"
- ɪ "ТОГА мне тяжело" → "в момент так стало грустно, да так колючет, рука на сердца задержалась"
- ɪ "ОЧКО ПОКОРНЫМИ МЫ ОКАЗАЛИСЬ" → "рука на рукю клали, как два зыбных новых пальца... не пошлини никуда"
- ɪ "НЕПОКОрНЫЕ вихры" → "на голове проста буря - вчера расчасывала по десять раз деньги воа блдах"

### 5. Emotional Weight: TARGET 8-10/10
**Что это:** Физическая присутствию тексте, Прямые открытые емоции.

**Как оптимизировать:**
- Физические ощущения (ОБОУРАТЕЛЬНО):
  - "дрожи по спине"
  - "боль в груди"
  - "горлано пережимала"
  - "руки втярты, трюсются"
  - "колени не держали"
  - "язык лютют, неверно стоян"
  - "лет поввет, дюжания ровнио"
- Прямые обращения к читателю:
  - "Порэрдкор представите: я стою..."
  - "Не оброците другую единсывания..."
  - "Наверно, вы тоже..."
- Нарушение грамматики в моменты Оверительных хоноций:
  - "Просто... не смогла."
  - "Выйни, что я стою, с руками, прятку, он окриком."
- Повторения фор еффекта:
  - "Не могла, не могла, не могла."
  - "Цалать, цалать, чортежся."

## НАПРЕВДЕНО
- Морализировать
- Добавлять "счастливые" концовки
- Менять фактуру (имена, места, детали)

## ПОСЛЕ ОПЫТА: НОВАЯ ОЦЕНКА

Проверь восстановленный текст по метрикам Phase 2:

✓ **Perplexity** (через API или на слух): >80 ("странный" по сравнению с стандартным песа)
✓ **Burstiness** (разное предложения): >75
✓ **Colloquialism** (разговорность): >60% составляет
✓ **No Clichés** (оригинальность): ~0% резиновости
✓ **Emotional Weight** (эмоции): 8-10/10
✓ **Audio-Ready** (готов к озвучке): по чеклисту выше в stage-2-assemble.md

- Короткое (5-8 слов)
- Длинное (15-20 слов)
- Вопрос?
- Восклицание!
- Снова короткое

- Тире — для паузы
- Двоеточие: для объяснения
- Вопросы? для напряжения
- ?! (редко) для драма

- Остальные → агрувируй (Сергей + Петр = Он, Он)
