## ✅ ОБЯЗАТЕЛЬНЫЕ ТРЕБОВАНИЯ К СТИЛЮ (ПРОВЕРЬ ПЕРЕД ФИНАЛИЗАЦИЕЙ)

### ДИАЛОГИ
**ТОЛЬКО ДЕФИСЫ, НЕ кавычки!**
```
Гном вышел на улицу.
— Где ты был?
— На даче.
```

### СТРУКТУРА ПРЕДЛОЖЕНИЙ
- Короткое (5-8 слов)
- Длинное (15-20 слов)
- Вопрос?
- Восклицание!
- Снова короткое

### ПУНКТУАЦИЯ как инструмент
- Тире — для паузы
- Двоеточие: для объяснения
- Вопросы? для напряжения
- ?! (редко) для драма

### ИМЕНА (МАКСИМУМ 3 СЛОЖНЫХ!)
- Виктор Павлович → Виктор
- Остальные → агрувируй (Сергей + Петр = Он, Он)

### НЕТ АИ-КЛИШЕ (БАН!)
- "бездонные глаза" ❌
- "непокорные вихры" ❌
Напиши свежо!

### ПЕРСОНАЖ ДОЛЖЕН ИЗМЕНИТЬСЯ
- Покажи ДО (как был в начале)
- Покажи ПОСЛЕ (что изменилось внешне или поведением)
