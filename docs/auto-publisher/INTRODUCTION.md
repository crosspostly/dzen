# Dzen Auto-Publisher - Полное руководство

## Введение

Dzen Auto-Publisher - это полностью автоматизированный инструмент для публикации статей из RSS-файла в Яндекс Дзен. Проект использует точные селекторы и правильную обработку HTML-тегов для надежной публикации контента.

## Основные возможности

- Автоматическая публикация статей из RSS-файла
- Автоматическая вставка изображений
- Имитация человеческого поведения
- Использование куки для аутентификации
- Обход простых систем детекции автоматизации
- Правильная обработка HTML-тегов в тексте
- Двухшаговая публикация (в редакторе и в модальном окне)
- Система отслеживания опубликованных статей (избегает дубликатов)
- Автоматическое сохранение истории публикаций

## Структура проекта

Проект состоит из минимального набора файлов, необходимых для работы:

- **src/main.js** - Основной скрипт публикации
- **modules/publication_history.js** - Модуль отслеживания публикаций
- **config/cookies.json** - Аутентификационные куки
- **data/published_articles.txt** - История публикаций
- **dzen-schema.json** - Точные селекторы для элементов
- **public/feed.xml** - RSS-фид с статьями
- **Документация** - Подробные файлы с инструкциями

## Начало работы

1. Установите Node.js
2. Установите зависимости: `npm install`
3. Установите браузер: `npm run install-browsers`
4. Подготовьте куки для аутентификации
5. Подготовьте RSS-фид с статьями
6. Запустите публикацию: `npm start`

## Безопасность

- Файл куки не должен публично распространяться
- Рекомендуется использовать отдельный аккаунт для автоматизации
- Не публикуйте слишком часто, чтобы избежать подозрений в автоматизации
- Следуйте рекомендациям в файле SECURITY_AND_RECOMMENDATIONS.md

## Поддержка и обновления

- Для ответов на частые вопросы смотрите FAQ.md
- При изменениях интерфейса Дзен следуйте инструкциям в UPDATE_INSTRUCTIONS.md
- Для автоматизации используйте инструкции в AUTOMATION.md

## Заключение

Этот проект предоставляет все необходимое для автоматической публикации статей в Яндекс Дзен. Следуйте инструкциям в документации, и вы сможете успешно использовать этот инструмент для автоматизации публикаций.
