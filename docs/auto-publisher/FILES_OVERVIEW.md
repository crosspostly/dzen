# Обзор всех файлов проекта

## Основные файлы

### src/main.js
Основной скрипт публикации, который:
- Читает статьи из feed.xml
- Открывает браузер и заходит в Дзен
- Заполняет заголовок и тело статьи
- Вставляет изображения
- Публикует статью в два этапа
- Сохраняет информацию о публикации

### modules/publication_history.js
Модуль для отслеживания публикаций, который:
- Читает и записывает историю публикаций
- Проверяет, была ли статья уже опубликована
- Предотвращает дублирование публикаций

### config/cookies.json
Файл с аутентификационными куки для доступа к Дзен

### data/published_articles.txt
Файл с историей опубликованных статей (дата и заголовок)

### dzen-schema.json
Файл с точными селекторами для взаимодействия с элементами Дзен

## Вспомогательные файлы

### public/feed.xml
RSS-фид с статьями для публикации (шаблон в feed.xml.example)

### feed.xml.example
Пример структуры RSS-файда с статьями

### package.json
Файл с зависимостями и командами для npm
