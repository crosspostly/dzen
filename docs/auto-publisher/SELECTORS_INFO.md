# Информация о селекторах

## Что такое dzen-schema.json

Файл `dzen-schema.json` содержит точные селекторы для взаимодействия с элементами интерфейса Яндекс Дзен. Эти селекторы критически важны для надежной работы автопублишера.

## Структура файла

Файл `dzen-schema.json` содержит массив объектов с информацией о селекторах, но в проекте также используются селекторы, определенные в основном скрипте.

## Основные типы селекторов

1. **Кнопка добавления публикации**: `[data-testid="add-publication-button"]`
2. **Кнопка "Написать статью"**: `text="Написать статью"`
3. **Кнопка вставки изображения**: `button.article-editor-desktop--side-button__sideButton-1z[data-tip="Вставить изображение"]`
4. **Первая кнопка публикации**: `[data-testid="publish-btn"]` (в редакторе)
5. **Вторая кнопка публикации**: `button[data-testid="publish-btn"][type="submit"]` (в модальном окне)

## Обновление селекторов

Если интерфейс Дзен изменится, и публикация перестанет работать, возможно, потребуется обновить селекторы:

1. Откройте DevTools в браузере при работе с Дзен
2. Найдите нужный элемент в DOM
3. Скопируйте точный селектор
4. Обновите соответствующие селекторы в основном скрипте

## Тестирование селекторов

Для проверки селекторов можно использовать консоль браузера:
```javascript
document.querySelector('ваш-селектор')
```

Если элемент найден, селектор работает правильно.
