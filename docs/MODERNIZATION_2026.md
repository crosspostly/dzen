# Модернизация Dzen Guru под стандарт 2026 (Апрель)

## Статус: ✅ ПОЛНОСТЬЮ ВНЕДРЕНО И ПРОТЕСТИРОВАНО

Проект успешно переведен на алгоритмы глубокого удержания (Deep Retention) и социального доказательства (Social Proof), актуальные для Дзена в апреле 2026 года.

### 🚀 Что нового:

1.  **Retention-Core (4500 знаков):**
    *   Все конфиги перенастроены на диапазон 3500-4500 знаков. Это максимизирует дочитываемость и доход от рекламы.
    *   Файлы: `BUDGET_CONFIG.ts`, `configService.ts`, `ContentFactory.ts`.

2.  **Confessional Skaz 2.0:**
    *   **Эффект запинки:** Имитация живой мысли («как бы это сказать...»).
    *   **Четвертая стена:** Прямые обращения к аудитории («Девочки, вы не поверите...»).
    *   **Физиология эмоций:** Замена эпитетов на телесные ощущения.
    *   Файл: `skazNarrativeEngine.ts`.

3.  **Smart Visual Hooks (Stage 5):**
    *   Автоматический анализ текста на «визуальный вес».
    *   Генерация обложки + 2 внутренних фото (мобильный стиль iPhone 15 Pro).
    *   Файлы: `visualHookService.ts`, `VisualHooks.ts`.

4.  **Auto-Relinking (Stage 6):**
    *   Автоматическая вставка ссылок на предыдущие статьи из `published_articles.txt`.
    *   Создает «бесконечную петлю» чтения на канале.
    *   Файл: `relinkingService.ts`.

5.  **Временные якоря:**
    *   Инъекция текущей даты и сезона («конец апреля», «весеннее настроение») для повышения доверия алгоритма.

### 🛠 Технические детали публикации:
*   Публикатор (`playwrightService.ts`) теперь поддерживает маркеры `[[FILE:path]]`.
*   Изображения загружаются **напрямую с диска** (не по URL), что гарантирует работу в GitHub Actions.
