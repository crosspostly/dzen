## Problem addressed

- Articles were getting corrupted by anti-detection and cleanup layers
- Pipeline felt too complex; users wanted a direct prompt approach
- Strict requirement: **first-person narration always**

---

## Key features

### 1) Simplified generation mode

New CLI flags:

- `--no-anti-detection` ‚Äî disables Phase 2 processing
- `--no-cleanup` ‚Äî skips cleanup gates


### 2) New ‚Äúsimple‚Äù generators

- `SimpleEpisodeGenerator`
  - one clean prompt
  - no Phase 2 / no cleanup
  - enforces first-person

- `SimpleArticleGenerator`
  - full article generation in one flow
  - designed for clean output without extra processing

### 3) Config wiring + backward compatibility

- New options in configs/options:
  - `useAntiDetection?: boolean`
  - `skipCleanupGates?: boolean`
- Full mode remains the default; no breaking changes
